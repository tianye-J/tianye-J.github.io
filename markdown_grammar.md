# Markdown 语法速查手册

写博客文章时的 Markdown 语法参考。

---

## 1. 文本样式

| 效果 | 语法 |
| --- | --- |
| **加粗** | `**文字**` |
| *斜体* | `*文字*` |
| ~~删除线~~ | `~~文字~~` |
| `行内代码` | `` `代码` `` |
| ***粗斜体*** | `***文字***` |

> **排版建议**：中文环境下优先使用**粗体**强调核心概念，少用*斜体*（中文斜体渲染效果一般）。

---

## 2. 标题层级

```markdown
# 一级标题（文章里一般不用，由 Front Matter 的 title 生成）
## 二级标题（章节划分）
### 三级标题（子模块）
#### 四级标题（很少用到，建议最多到三级）
```

---

## 3. 段落与换行

```markdown
段落之间用一个空行分隔。

这是第二段。直接回车不会换行，
需要在行尾加两个空格再回车  
才能强制换行（但更推荐用空行分段）。
```

---

## 4. 列表

**无序列表**——用 `-` 或 `*` 开头：

```markdown
- 第一项
- 第二项
  - 嵌套子项（缩进两个空格）
  - 另一个子项
```

**有序列表**——用数字 `1.` 开头：

```markdown
1. 第一步
2. 第二步
3. 第三步
```

**任务清单**——用 `- [ ]` 和 `- [x]`：

```markdown
- [x] 已完成的任务
- [ ] 未完成的任务
```

---

## 5. 引用

用 `>` 开头，可嵌套：

```markdown
> 这是一段引用。适合写摘要、核心观点或他人原话。
>
> > 嵌套引用

> **注意**：引用块内可以使用其他 Markdown 语法。
```

---

## 6. 代码

**行内代码**：用反引号包围。

```markdown
请运行 `hugo server -D` 启动本地预览。
```

**代码块**：用三个反引号包围，标注语言名可获得语法高亮：

````markdown
```python
import torch

model = torch.nn.Linear(10, 1)
print(model)
```
````

支持的常用语言标注：`python`、`cpp`、`bash`、`go`、`javascript`、`yaml`、`toml` 等。

---

## 7. 数学公式（LaTeX）

> 需要在 Front Matter 中添加 `math = true` 才能启用渲染。

**行内公式**：用单个 `$` 包围。

```markdown
质能方程 $E=mc^2$ 是物理学中最著名的公式。
```

**独立公式块**：用 `$$` 包围，会单独居中显示。

```markdown
$$
\mathcal{L} = \alpha L_{\text{cls}} + (1-\alpha) L_{\text{reg}}
$$
```

常用 LaTeX 符号速查：

| 写法 | 效果 |
| --- | --- |
| `\frac{a}{b}` | 分数 |
| `x^{2}` | 上标 |
| `x_{i}` | 下标 |
| `\sum_{i=1}^{n}` | 求和 |
| `\int_{a}^{b}` | 积分 |
| `\sqrt{x}` | 平方根 |
| `\boldsymbol{x}` | 粗体向量 |
| `\hat{x}`, `\bar{x}`, `\dot{x}` | 修饰符 |

## 8. 链接与图片

**超链接**：

```markdown
[GitHub 主页](https://github.com/your-username)
```

**图片**：

```markdown
![图片描述](/images/example.png)
```

> Hugo 中图片放在 `static/images/` 目录下，引用路径从 `/images/` 开始（不写 `static`）。

**带标题的图片（Hugo figure shortcode）**：

```text
{{< figure src="/images/example.png" caption="图 1：系统架构" >}}
```

---

## 9. 表格

```markdown
| 左对齐 | 居中对齐 | 右对齐 |
| :--- | :---: | ---: |
| 内容 | 内容 | 内容 |
| 内容 | 内容 | 内容 |
```

用 `:` 的位置控制对齐方式：`:---` 左对齐、`:---:` 居中、`---:` 右对齐。

---

## 10. 分割线

三个或以上的 `-`、`*` 或 `_`：

```markdown
---
```

---

## 11. 脚注

```markdown
这里有一个脚注引用[^1]，还有一个[^note]。

[^1]: 这是脚注的内容。
[^note]: 脚注标识可以用数字或英文单词。
```

脚注内容会自动汇集到页面底部，适合标注参考文献或补充说明。

---

## 12. 转义字符

当你需要显示 Markdown 语法符号本身时，在前面加 `\`：

```markdown
\* 这不会变成列表项
\# 这不是标题
显示反引号：\`代码\`
```

---

## 13. HTML 嵌入

Markdown 中可以直接写 HTML，适合需要精细控制的场景：

```html
<details>
<summary>点击展开详细内容</summary>

这里是折叠的内容，支持 **Markdown** 语法。

</details>

<!-- 居中显示 -->
<div align="center">居中文字</div>
```

---

## 14. Hugo 首页截断符

在文章中插入 `<!--more-->` 标记，首页只显示该标记以上的内容，并出现"阅读更多"按钮：

```markdown
这是文章开头的摘要部分...

<!--more-->

这里是正文的后续内容，不会出现在首页列表中。
```
